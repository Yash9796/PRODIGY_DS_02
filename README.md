# PRODIGY_DS_02 – Data Cleaning and Exploratory Data Analysis (EDA)

## 📌 Objective
The objective of this task is to perform data cleaning and exploratory data analysis (EDA) on a dataset (Titanic dataset from Kaggle) to explore relationships between variables and identify patterns and trends in the data.

## 🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Dataset Used
Titanic Dataset – Kaggle

The dataset contains information about passengers such as:
- Age
- Gender
- Passenger Class (Pclass)
- Fare
- Survival Status
- Embarked Port

## 🧹 Data Cleaning Steps
1. Checked for missing values
2. Handled missing data (Age, Cabin, Embarked)
3. Removed unnecessary columns
4. Converted categorical variables into appropriate format
5. Verified data types

## 📈 Exploratory Data Analysis (EDA)
The following analyses were performed:

- Survival rate comparison between male and female passengers
- Impact of passenger class on survival
- Age distribution of passengers
- Fare distribution
- Correlation analysis between numerical variables
- Visualization using bar plots, histograms, and heatmaps

## 📊 Key Insights
- Female passengers had a higher survival rate compared to males.
- Passengers in 1st class had higher survival probability.
- Younger passengers had slightly better survival chances.
- Passenger class and fare showed a strong relationship.
